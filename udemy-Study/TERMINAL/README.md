## 터미널이란?
- vsCode에서는 파일 생성, 삭제를 GUI(버튼 클릭)를 이용하여 가능하다
- 터미널은 CUI를 이용하여 가능

### 터미널을 사용하면 좋은이유
- 개발 효율이 좋아진다.
  - 커맨드에 적응하면 작업속도가 상당히빨라진다.
- 엑세스범위
  - 평소에 볼수없는 부분까지 엑세스하는것이 가능
- Node.js, DB 를 사용할때 터미널을 사용한다.
  - Node.js에서 서버를 열거나할때 반드시 터미널을 거친다.

### 쉘
- 터미널상에 움직이는 프로그램
- ATM이 터미널 ATM에서 작동하는 프로그램이 쉘
- 다양한 쉘을 사용하는것이 가능
